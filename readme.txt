Easily embed a trip report index (or other list of commonly tagged posts) into another post

== Description ==
Have a series of posts with a common tag? Create a list of them embedded in another post. Great for a footer showing the index of a series, updated dynamically. Also an option to include links just to the previous and next posts in a series rather than the full list. 

More details available on the [plugin homepage](http://blog.wandr.me/plugins/tag-lister/).

== Installation ==

1. Upload `tag-lister.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[taglist]` or `[tagprevnext]` entries in your posts or pages, using the syntax defined on the help page
